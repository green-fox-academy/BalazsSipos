F1 = Command Palette

C+k+s = list of commands

A+arrows
= Move line up/down

S+A+arrows
= Copy line up/down

C+S+k = Delete line

C+Enter = insert line below

C+arrows
= Scroll window (like mouse wheel)

C+k+c = Add line comment

C+k+u = Remove line comment

C+l = Select current line

C+F2 = Select all occurences of current word

C+Space = Trigger suggestion

S+A+f = Format document

C+k+t = Theme selector
